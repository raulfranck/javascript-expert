# Package.json

## --experimental-top-level-await
No script "dev" precisa incluir a flag --experimental-top-level-await para que o node entenda a syntaxe de utilizar await fora de async

Ex: arquivo index.js

para que isso funcione o node precisa estar na versão 14.3.0 ou sup

## --experimental-json-modules
Para poder importar arquivo json no javascript

## engines
Define no arquivo package.json qual a versão necessária para o node.
